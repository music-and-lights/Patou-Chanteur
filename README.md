# Patou Chanteur — React App

Application React (Vite) du site **Patou – Artiste Chanteur Musicien**.

## Scripts

- `npm install` : installe les dépendances
- `npm run dev` : démarre le serveur de développement
- `npm run lint` : exécute ESLint
- `npm run build` : génère le build de production
- `npm run preview` : prévisualise le build localement

## Structure

- `src/siteMarkup.html` : markup du site (contenu/copy d’origine)
- `src/siteStyles.css` : styles d’origine
- `src/App.jsx` : rendu React + comportement navbar au scroll
